About this web application

This basic web application is designed as a simple learning project for beginners to understand how HTML, CSS, and JavaScript work together to build interactive websites. It provides a To-Do list where users can add, complete, or delete tasks, with everything saved directly in the browser using localStorage. It also includes a demo contact form to practice handling user input. The goal of this app is to give learners a hands-on starting point for exploring web development while building a functional and easy-to-use tool.

Features

  • Clear semantic structure (header, main, footer)
  • Navigation links that jump to sections
  • Responsive layout using CSS grid (works on phones and desktop)
  • Contact area that opens your email client (mailto link)
  • All content in a single HTML file — easy to edit and share
